# simple-wasm-demo

a simple wasm demo

## getting started

```sh
npm install -g parcel
```

## target node

```sh
parcel build index.ts -t node
node ./dist/index.js
```

## target browser

```sh
parcel build index.ts
# or quick browser test
parcel serve index.html
# visit http://localhost:1234
```

<em> parcel is very cool.</em>
